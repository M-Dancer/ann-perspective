# Andrej Karpathy · 著作与系统性思考

## Software 2.0（软件2.0）

**最重要的文章**（2017年，karpathy.ai）
核心论点：神经网络不只是机器学习工具箱里的另一种分类器，而是软件开发范式的根本转变——Software 2.0。

核心对比：
| 软件1.0 | 软件2.0 |
|---------|---------|
| 人类写明确规则 | 人类定义损失函数，机器学习规则 |
| 代码是产品 | 数据是产品 |
| 调试：改代码→看行为 | 调试：改数据→看行为变化 |
| 可解释 | 不可解释（黑箱） |

**跨域验证**：Software 2.0概念在他讨论的所有场景中出现：
- 自动驾驶（Policy Network → 端到端学习）
- AI教育（课程内容 = 训练数据，AI = 学习模型）
- 通用编程（Copilot = 下一个token预测）
- 2024年的「100% Fully Software 2.0」构想

**自创术语**：
- Software 2.0 / Software 1.0
- "data is code" / "code is data"
- "alchemy"（对神经网络训练中经验性/缺乏理论解释的描述）
- "loss landscape"
- "gradients"

## 核心课程与教育哲学

**CS231n（斯坦福卷积神经网络课程）**：
- 全球最受欢迎的深度学习课程之一
- 核心理念：「从第一性原理教懂每一个人」
- 教学风格：极度耐心、类比丰富、重视直觉而非公式

**makemore系列**：
- GitHub: karpathy/makemore
- 从零复现GPT-2（4小时视频，1000行C代码实现GPT-2训练）
- 核心理念：「理解神经网络的最好方式，是从零复现它」
- 教育哲学：动手做胜过看理论

**nanoGPT**：
- GitHub: karpathy/nanoGPT
- 最简GPT实现，易读性极高
- 使命：「让GPT不再神秘」

## Eureka Labs（2024年）

**创办理念**：
- 「教师 + AI」的共生教育模式
- 人类专家写课程材料，AI助教指导学生
- 类比：「想象你和费曼一起学物理，有高质量课程材料，他会指导你每一步」

**第一款产品：LLM101n**：
- 本科水平AI课程
- 让学生训练自己的LLM
- 核心理念：「学AI的最好方式是做AI」

## 核心信念（反复出现≥3次）

1. **「从零理解」**：最好的理解来自从零复现，不是看教程
2. **「AI是新一代编程范式」**：神经网络是软件2.0，不是ML工具箱
3. **「数据优先」**：模型不行时，首先检查数据
4. **「教育民主化」**：AI可以让80亿人获得高质量教育
5. **「神经网络是炼金术」**：我们还不理解为什么它有效，但经验上有效
